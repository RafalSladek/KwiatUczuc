# Kwiat Uczuć — strategia produktu i marketingu

## 1. TL;DR

- Produkt: tracking emocji (dzieci + dorośli)
- Core value: lepsze rozmowy + wsparcie terapii
- Branding: kwiat / koło emocji
- Nazwa: **Kwiat Uczuć**
- Strategia: niche + SEO + psychologowie + viral w produkcie

---

## 2. Problem

- Rynek: bardzo konkurencyjny (habit tracker / journaling)
- Brak jasnego positioning = brak użytkowników
- „Kontakt" jako nazwa → nie działa

---
